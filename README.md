# OneDocs

OneDocs is a modern, feature-rich document editor inspired by Microsoft Word and Google Docs. Built with React and Tailwind CSS, it delivers high performance and a smooth user experience.

## ‚ú® Key Features

- **Ribbon-style Interface**: A familiar toolbar with tabs (File, Home, Insert, Layout, etc.) for intuitive navigation.
- **Smart Pagination**: Automatically adds new A4 pages as content fills up, ensuring a seamless writing flow.
- **Local File System Access**: Directly open and save files in `.docx`, `.html`, and `.md` formats (utilizing the File System Access API).
- **Voice Typing**: Built-in dictation support powered by the Web Speech API.
- **Professional Templates**: Quick start with ready-made templates for resumes, invoices, and meeting minutes.
- **Dark Mode**: Full Dark Mode support for comfortable work at any time of day.
- **Focus Mode**: Hides UI distractions, allowing you to focus entirely on your text.
- **Print & Preview**: Optimized styles for printing documents perfectly.
